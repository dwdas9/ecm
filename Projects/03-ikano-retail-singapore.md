# IKANO Retail (An IKEA Company)
**Role:** Assistant Manager  
**Platform:** OpenText Intelligent Capture 21.4 · Azure Databricks

---

## Overview

Built APIs for Captiva validation and conducted proof-of-concept using AI Builder to evaluate reduction of Captiva licensing costs. Modernization initiative alongside Azure data engineering workstreams.

---

## Tech Stack

- **OpenText IC 21.4:** XML Export, Standard Import, Dispatcher
- **Azure:** App Service, ADLS, ADF, DevOps, Key Vault, Entra ID, Logic Apps
- **Analytics:** Azure Databricks, Lakehouse Platform
- **Modernization:** Microsoft AI Builder, Power Automate, Canvas Apps
- **Languages:** C#.NET, Python, PowerShell

---

## Key Achievements

- **Data Integration:** Integrated XML output from Captiva Standard Export module into ADLS for downstream Databricks Lakehouse analytics processing
- **API Development:** Developed Python-based REST APIs on Azure App Service for document metadata validation
- **Inter-system Integration:** Worked with INGKA team for integration of Captiva processes with INGKA APIs
- **Infrastructure Management:** Developed PowerShell scripts for environment health checks and Azure cost monitoring
- **Cost Optimization:** Led technical evaluation and phased replacement of Captiva CaptureFlows with Microsoft AI Builder and Power Automate workflows to reduce licensing costs

---

## Impact

Successfully modernized capture infrastructure with reduced third-party scanning vendor dependency and evaluated AI Builder as a cost-effective alternative to OpenText Captiva licensing.
