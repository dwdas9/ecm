# HCL/Citibank AU - Project OASIS
**Role:** Software Engineer / Lead Software Engineer  
**Platform:** OpenText Captiva InputAccal 5.x/6.x · Documentum 5.x/6.x  
**Parent Organization:** HCL Technologies (Singapore Hub)

---

## Overview

Citibank Australia planned to introduce a digital Workflow and Imaging Solution (W&I) across its branches to create a more efficient, nearly paperless system for managing back-office banking operations. This system aimed to automate business processes, reducing costs, increasing employee productivity, ensuring accuracy, and enhancing management oversight.

---

## Tech Stack

- **OpenText:** Documentum 5.3/6.x (Content Server, DA, Composer, Webtop/WDK, DFC, BPM), Captiva InputAccal 5.x/6.x (InputAccal Server, Administrator Console, ScanPlus, IndexPlus, Image Enhancement, Watch/MDW, Multi, ODBC Export, Documentum Export, eInput/eScan/eIndex, Dispatcher, ScaleServer, ClickOnce)
- **Infrastructure:** MSCS Active-Active/Active-Passive, SQL Server 2005/2008, Windows Server 2003/2008, IIS 6/7
- **Languages:** C#.NET, VB6, VBA, Java

---

## Key Achievements

- **Branch-Wide Rollout:** Implemented digital Workflow and Imaging Solution across Citibank AU branches
- **Process Automation:** Automated back-office banking operations with nearly paperless workflows
- **Infrastructure Support:** Worked with EMC, HCL, and network teams for Captiva and Documentum deployment across environments (Dev/SIT/UAT/Prod, MSCS, ScaleServer)
- **CaptureFlow Development:** Built Captiva workflows (Watch, Scan, Index, Multi, Dispatcher Classification/Recognition/Extraction, Image Enhancement, eInput, Documentum Export)
- **Backend Integration:** Built backend integrations with ODBC Oracle connections, SOAP services, and stored procedure validation
- **Client Deployment:** Rolled out eInput (eScan, eIndex) across branches via Microsoft ClickOnce
- **Production Support:** Provided L3 production support for batch failures, recognition issues, export failures, and workflow issues

---

## Impact

Successfully transformed Citibank Australia's back-office banking operations from paper-based to nearly-paperless system, automating business processes, reducing costs, and improving employee productivity across multiple branches.
