# HCL/Citibank - Project CGSL (HR Recruitment)
**Role:** Software Engineer / Lead Software Engineer  
**Platform:** OpenText Documentum 5.x/6.x (Webtop Customization) · Captiva 5.x/6.x  
**Parent Organization:** HCL Technologies (Singapore Hub)

---

## Overview

HR Recruitment Application implemented as Webtop customization to automate the HR recruitment process and digitize documents of Citigroup Global Services Limited. The solution captures documents through the scanner using EMC Documentum and EMC Captiva for capture, processing, storage, and retrieval.

---

## Tech Stack

- **OpenText:** Documentum 5.x/6.x (Content Server, Composer, Webtop/WDK - customized, DFC, BPM), Captiva InputAccal 5.x/6.x (ScanPlus, IndexPlus, Image Enhancement, Multi, ODBC Export, Documentum Export, eInput/eScan/eIndex)
- **Development:** C#.NET, VB6, Java
- **Infrastructure:** Windows Server 2003/2008, SQL Server 2005/2008, Oracle 9i/11g

---

## Key Achievements

- **Webtop Customization:** Implemented HR Recruitment Application as Webtop customization
- **Process Automation:** Automated the HR recruitment process end-to-end
- **Document Digitization:** Digitized all employee recruitment documents
- **Multi-Channel Capture:** Enabled document capture through scanners
- **Integration:** Integrated with backend ECM system for processing, storage, and retrieval
- **Workflow Management:** Implemented document lifecycle and approval workflows
- **User Portal:** Developed Webtop-based portal for HR users

---

## Impact

Transformed Citigroup's HR recruitment process from manual paper-based system to automated digital workflow, improving recruitment efficiency, document organization, and process transparency.
