# HCL/Citibank SG - Project NorthStar
**Role:** Software Engineer / Lead Software Engineer  
**Platform:** OpenText Captiva InputAccal 5.x/6.x · Documentum 5.x/6.x  
**Parent Organization:** HCL Technologies (Singapore Hub)

---

## Overview

NorthStar Application automates and streamlines various banking processes for Citibank Singapore branches. Transitioning the current account opening processes to a paperless format using Captiva and Documentum. Integrated all Citibank Singapore branches with the newly developed system.

---

## Tech Stack

- **OpenText:** Documentum 5.3/6.x (Content Server, DA, Composer, Webtop/WDK, DFC, BPM, DQL), Captiva InputAccal 5.x/6.x (Administrator Console, ScanPlus, IndexPlus, Image Enhancement, Multi, ODBC Export, Documentum Export, eInput/eScan/eIndex, Dispatcher Classification/Recognition, ScaleServer, ClickOnce)
- **Infrastructure:** MSCS Active/Passive, SQL Server 2005/2008, Oracle 9i/11g, Windows Server 2003/2008, IIS 6/7
- **Languages:** C#.NET, VB6, Java, VBA

## Key Achievements

- **Paperless Transformation:** Transitioned account opening processes from paper-based to completely paperless format
- **Multi-Product Support:** Automated account opening for 9+ product lines
- **CaptureFlow Development:** Designed and built Captiva workflows for document capture and processing
- **Backend Integration:** Integrated with core mainframe systems and Documentum repository
- **Branch Integration:** Integrated all Citibank Singapore branches with the new system
- **Classification & Recognition:** Implemented Dispatcher classification and recognition for different product types
- **Metadata Extraction:** Built OCR-based extraction for key data fields
- **Approval Workflows:** Configured document approval and release workflows

## Impact

Successfully implemented first paperless account opening system at Citibank Singapore branches, automating account opening across 9+ product lines and enabling same-day processing for customer accounts.
