# TCS Canada - Federal Crown Corporation Captiva Upgrade
**Role:** Solution Architect / Project Manager  
**Organization:** Canadian Federal Government Corporation  
**Platform:** OpenText Documentum 6.5→7.2 · OpenText Captiva 6.5→7.5

---

## Overview

On-site engagement for a Canadian federal government corporation. Travelled extensively across Canada for requirements gathering and on-site delivery. Led Documentum and Captiva platform upgrade and modernization initiatives.

---

## Tech Stack

- **OpenText:** Documentum 6.5→7.2, Captiva 6.5→7.5 (ScanPlus, IndexPlus, Image Enhancement, Image Converter, Multi, ODBC Export, Documentum Advanced Export, eInput, Dispatcher, ScaleServer, ClickOnce)
- **Infrastructure:** Windows Server 2012 R2/2016, SQL Server 2012-2016 (AlwaysOn, MSCS)
- **Cloud:** Azure VM, Azure SQL, AzCopy
- **Languages:** C#.NET, IIS 8.5

---

## Key Achievements

- **Platform Upgrade:** Led upgrade of OpenText Documentum and Captiva infrastructure from legacy versions to current generation
- **Enhanced Processes:** Enhanced and upgraded existing Captiva processes replacing deprecated modules and adding new functionalities
- **API Integration:** Created .NET assembly proxy to consume SOAP web services for metadata validation
- **Data Validation:** MSSQL stored procedures for data validation and reporting
- **Vendor Management:** Managed OpenText vendor relationship, licensing repackaging, and third-party coordination
- **Requirements Gathering:** Travelled across Canada for on-site requirements gathering and delivery
- **Multi-Site Coordination:** Coordinated deployment across multiple federal locations

---

## Impact

Successfully modernized federal government corporation's ECM infrastructure, enabling secure, compliant document management and capture for sensitive government operations.
