# Inland Revenue Authority of Singapore (IRAS)
**Role:** Senior Manager & Solution Architect  
**Platform:** OpenText Intelligent Capture 22.4 · Azure Government Commercial Cloud (GCC)

---

## Overview

Led paperless office automation and intelligent document capture initiatives on Azure Government Commercial Cloud (GCC) handling sensitive tax data. Collaborated with multiple government agencies on data integration as part of a broader whole-of-government digital transformation program.

---

## Tech Stack

- **OpenText IC 22.4:** Designer, Administrator, Standard Import, Image Processor, Multi, .NET Code, Documentum Export
- **Azure:** App Service, File Share, ADLS, Key Vault, Entra ID, VNet, Private Endpoints, Managed Identities
- **Infrastructure:** MSSQL on Azure VM (IaaS), Docker
- **Languages & Tools:** C#.NET, Python, PowerShell, REST APIs (ASP.NET Core Web API)

---

## Key Achievements

- **Government Integration:** Worked with government shared services group to integrate Individual Income Tax and Property Tax Division data via APIs into Captiva CaptureFlows
- **Multi-Agency Collaboration:** Worked with GovTech and Property Tax division on their initiative to integrate their intelligent document processing with IRAS through APIs
- **API Development:** Built .NET REST APIs on Azure App Service for MSSQL-based data validation on Captiva
- **Data Pipeline:** Integrated XML output from Captiva Standard Export into ADLS for downstream tax analytics
- **Security & Compliance:** Configured RBAC across Azure VMs, SQL Server, and APIs; participated in compliance audits
- **L3 Support:** Provided production support and developed PowerShell scripts for VM health checks and service monitoring
- **Infrastructure Management:** Oversaw provisioning of Azure resources - resource groups, ADLS, VNets, private endpoints, NSGs, managed identities, and Key Vault

---

## Impact

Enabled paperless tax processing across multiple government divisions while maintaining strict compliance and security requirements for sensitive taxpayer data on Azure GCC infrastructure.
