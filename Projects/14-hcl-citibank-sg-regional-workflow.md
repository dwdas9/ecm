# HCL/Citibank SG - Regional Data Extraction & Workflow Solution
**Role:** Software Engineer / Lead Software Engineer  
**Platform:** OpenText Captiva InputAccal 5.x/6.x · Documentum 5.x/6.x  
**Parent Organization:** HCL Technologies (Singapore Hub)
**Geographic Scope:** 5 Countries (Malaysia, Philippines, Indonesia, Thailand, Guam)

---

## Overview

Regional Build is an Imaging & Workflow Solution (I&W) implemented across all branches and offices of Citibank in five countries to achieve near-paperless office automation for banking services back-office processes.

---

## Tech Stack

- **OpenText:** Documentum 5.3/6.x (Content Server, DA, Composer, Webtop/WDK, DFC, BPM, DQL), Captiva InputAccal 5.x/6.x (InputAccal Server, Administrator Console, ScanPlus, IndexPlus, Image Enhancement, Watch/MDW, Multi, ODBC Export, Documentum Export, eInput/eScan/eIndex, Dispatcher Classification & Recognition, ScaleServer, ClickOnce)
- **Infrastructure:** MSCS Active-Active/Active-Passive Clustering, SQL Server 2005/2008, Oracle 9i/11g, Windows Server 2003/2008, IIS 6/7, Apache Tomcat
- **Languages:** C#.NET, VB6, Java, VBA

---

## Solution Functionalities

- **Document Capture** — Multi-channel ingestion from scanners, network folders, email
- **Processing and Storage** — Image enhancement, OCR, classification, validation
- **Retention and Auditing Policies** — Document lifecycle management and compliance tracking
- **Records Management** — Retention schedules, destruction, legal holds
- **Workflows** — Approval workflows, task routing, SLA management
- **Role Management** — User access control, permission management, audit trails
- **Email Integration** — Capture and index email documents

Successfully achieved near-paperless office automation across Citibank's banking services back-office operations spanning 5 ASPAC countries, processing millions of documents annually with standardized workflows, compliance tracking, and records management.
